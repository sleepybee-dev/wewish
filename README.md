# 위위시wewish
<img src="https://img.shields.io/badge/version-0.0.1-blue"/>
나의 위시리스트를 관리하고 공유해 보세요.

## 주요 기능
- 타 앱으로부터 공유 기능을 통한 간편한 위시 추가 및 관리
- dynamic link를 활용한 위시리스트 공유
- 별도의 가입없이 다른 사람의 위시리스트 검색
